# Mdworld Stox demo

Based on:
A tiny demo using the [einaros/ws](http://einaros.github.io/ws/) WebSockets implementation.

# Running Locally

``` bash
npm install
npm start
```
or instead of npm start, run

``` bash
foreman start
```

# Running on Heroku

``` bash
heroku create
git push heroku master
heroku open
```
